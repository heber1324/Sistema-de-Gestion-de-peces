# 🐠 Sistema de Gestión de Peces

Este es un proyecto de catálogo de acuario desarrollado con Python (Flask) y SQLAlchemy. 

Características
- CRUD Completo: Registro y eliminación de los peces en tiempo real.
- Base de Datos: Persistencia de datos mediante SQLite para mantener los datos ya regitrados dentro del registro.
- Diseño Premium: Estilo Dark Mode utilizando Bootstrap 5.
- Animaciones: Efectos de scroll mediante la librería AOS.

Tecnologías utilizadas
- Python 3
- Flask
- Flask-SQLAlchemy (Base de datos)
- Jinja2 (Templates)
- AOS Library (Animaciones)
